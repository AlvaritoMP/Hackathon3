Etiqueta "body" representa el contenido principal de un documento HTML. Solo hay un elemento body en un documento


Etiqueta "nav" define una sección que solamente contiene enlaces de navegación, Ejemplo:
  • Inicio
  • Nosotros
  • Contacto

Etiqueta "br" genera espacios en blanco en el navegador entre las etiquetas del codigo

Etiqueta "article" define contenido autónomo que podría existir independientemente del resto del contenido. Lorem ipsum dolor, sit amet consectetur adipisicing elit. Etiqueta "aside" define algunos contenidos vagamente relacionados con el resto del contenido de la página. Se puede utilizar para poner anuncios o información a los lados de la pagina.

Etiqueta "address" define una sección que contiene información de contacto.

Etiqueta "footer" define el pie de una página o sección. Usualmente contiene un mensaje de derechos de autoría, algunos enlaces a información legal o direcciones para dar información de retroalimentación.
